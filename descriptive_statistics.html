3 Variables and Data Types

In the “penguins” dataset, there are 8 columns. The columns “species”, “island”, and “sex” contain categorical data. I believe the column “year” also could be counted as categorical as it’s not being used as a measurement of time, and it’s just listing the year of the study. The columns “bill_length_mm”, “bill_depth_mm”, “flipper_length_mm”, and “body_mass_g” all contain quantitative, continuous data. It doesn’t appear that “flipper_length_mm” and “body_mass_g” contain any non-integer values, however, because they are measurements in millimeters and grams, they are not discrete data. This matters because, in order for descriptive statistics to be meaningfully applied, a data analyst must be aware of the type of data being analyzed and what types of analysis would make sense for it.

The most common value in the `island` column is Biscoe. I chose `island` for `count()` because it is a categorical variable, so it makes sense to count the number of observations per island. This mode tells me that there were more penguins sampled from Biscoe island than Dream or Torgersen islands.

5 Measures of Center

5.1 Median

Goal: Identify a suitable variable from the “penguins” dataset for calculating the median.

median(penguins$bill_length_mm, na.rm = TRUE)
[1] 44.45
The median of the "bill_length_mm" column in the `penguins` dataset is 44.45.

The bill_length_mm variable is a quantitative, continuous variable, so it makes sense to determine the median. The median of these measurements is 44.45 mm, which means that it is the middle value of the variable. The median is a useful measure of center, as it is not sensitive to outliers, and it can provide a more accurate estimate of centrality when there are outliers in a dataset.

\[ \text{Range} = x_{(n)} - x_{(1)} \]

The range of a dataset is a description of the overall spread of the numbers in the dataset. It is easy to calculate and understand, but it only uses two values. Because of that, outliers can make the data seem more spread out than it truly is.
